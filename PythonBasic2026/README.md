# Python入門

### Hello World!

この講座ではプログラミング言語「**Python**」の基礎的な構文を学習します。Pythonの特徴として、コードの記述が比較的シンプルというのがあります。プログラミングを始めるときにまずはここから入る人もたくさんいます。

この講座を修了するころにはPythonを用いたプログラミングの基礎が身についているはずです。一緒に頑張りましょう！

## 目次

[1.環境構築(プログラミングをするための準備)](./1.install)  
プログラミングをするのに必要なものをインストールします。数研の部室でこれを学習している場合は完了している場合が多いです。

[2.Hello World!](./2.Hello_World!)  
初めの一歩となる文字列の出力を学びます。

[3.Variable & Function](./3.Variable%20%26%20Function)  
プログラミングで非常に重要な概念である変数と関数を学びます。

[4.Calculation](./4.Calculation)
足し算や引き算などの基本的な演算を扱います。

[5.Bool and If](./5.Bool%20and%20If)
データ型の一つであるboolと、非常に大切な構文のIf文を学びます。

[6.For and While](./6.For%20and%20While)
これもまた非常に大切な繰り返し処理を扱う、for文とwhile文を学びます。

[7.List,Tuple,Dictionary](./6.For%20and%20While)
やっぱりこれも非常に大切なデータ構造のlist、tuple、dictionaryを扱います。

ここまで出来たらPythonの入門編は完了です。
