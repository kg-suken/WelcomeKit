# 数値と演算子

この章ではPythonで計算をする方法を扱います。計算には**演算子**というものを使います。

| 記号 | 計算 |
----|---- 
| **+** | **足し算** |
| **-** | **引き算** |
| * |**掛け算**|
| **/** |**割り算**|
| **%** |**あまり**|
| **//** |**割り算(小数点以下切り捨て)**|

ここで、データの**型**というものを知っておく必要があります。

| 型の名前(略称) | 役割 |
----|---- 
| **string(str)** | **文字列** |
| **integer(int)** | **整数** |
|**float**|**浮動小数点**|

ちなみに、`/`(割り算)の計算結果はfloat型に、`//`(切り捨て)の計算結果はint型になります。また、`3/0`や`5//0`のように$0$で割るとエラーが出てしまうのでしないようにしましょう。

例
```python
# 5+9
print(5+9)

# 5-8
print(5-8)

# 3×8
print(3*8)

# 5÷3
print(5/3)

# 20を7で割った余り
print(20%7)

# 20÷3の商
print(20//3)
```

実行結果
```python
14
-3
24
1.666666666
6
6
```
